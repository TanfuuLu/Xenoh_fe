# Xenoh — Marketing UI kit

Public-facing pages for `xenoh.app`. Built to the same visual
foundations as the web app but with bigger display type, more
sand/paper surfaces, and generous vertical rhythm.

## Components

| File              | What it is                                          |
|-------------------|-----------------------------------------------------|
| `MarketingNav.jsx`| Top nav (logo, links, Sign in / Start)              |
| `Footer.jsx`      | Footer with the four link columns                   |
| `Hero.jsx`        | Full-bleed sand hero with display headline          |
| `Feature.jsx`     | Text-left / visual-right feature section            |
| `PricingCard.jsx` | Individual / Coach pricing tiers                    |
| `Testimonial.jsx` | Pull-quote with author + role                       |
| `LogoBar.jsx`     | "Used by coaches at…" logo row                      |
| `CTA.jsx`         | Closing full-bleed CTA band                         |
